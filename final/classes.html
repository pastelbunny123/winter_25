Classes!

Here is a list of all my classes, and what I like about them!

My first class is Yoga 7/8. I enjoy this class because I get to improve my strength and flexibility.

My second class is Algebra 2. In this class, I have been working on my final math presentation, where we complete a few problems and present our problem-solving process to our class.

My third class is Spanish 2. I enjoy this class because I get to learn more about the language, and I feel like I have gotten more proficient in both writing and speaking Spanish. In this class, I enjoy singing songs, writing stories, and doing group presentations.

My fourth class is Study Hall. I like having time to do assignments and work on projects during this time.

My fifth class is WEB. I enjoy this class because I learn new coding skills and how to build a website. I am currently working on my final project, this website!

My sixth class is Literary Thinking 3. Currently, I am working on my final, writing a LitShop letter. In this class, we have read Life of PI, The Hunger Games, and an individual reading LitShop book. Next class, I will be participating in the EPS Chow Games, our classes hunger games simulator!

My seventh class is Scientific Thinking 3. So far, we have learned about astronomy and light, along with other topics. I enjoy learning new things about the Earth in this class, and participating in labs.

My final class is Historical Thinking 3, where I have learned about Hinduism, Sikhism, Buddhism, Hinduism and Judaism.In the future, we will also be learning about Christianity and Islam, and visiting different buildings specific to each religion.
